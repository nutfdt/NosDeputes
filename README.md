# Android NosDéputés.fr 

## Explication du projet

Ce projet est une application Android qui permet de récupérer des informations sur les députés français au travers de [l'API de NosDeputés.fr](https://github.com/regardscitoyens/nosdeputes.fr/blob/master/doc/api.md) et ainsi de consulter celles-ci pour avoir diverses informations telles que les noms des collaborateurs du député ou encore son adresse e-mail.


## Captures d'écran

* On y retrouve la page principal où l'on recherche le ou les députés par leur nom, leur groupe parlementaire, département ou code postal

* Ensuite en sélectionnant le député voulu on se retrouve sur sa page où l'on a accès à plusieurs informations tels que son adresse e-mail, ces collaboratteurs, l'adresse de sa permanence, etc...

<img src="https://github.com/nutfdt/NosDeputes/assets/49862152/b609aa45-7afc-472e-89c0-96520ab39b81" width="300" height="600">


* Et pour finir si l'on souhaite visualiser les lois votées par le député, le bouton "Voir les votes" permet d'afficher celles-ci et nous montre aussi des informations tels que le nombre de votants ou encore le résultat du vote.
